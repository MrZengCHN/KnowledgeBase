**codex 默认是使用的cmd读取文件，低效且错误多，速度慢**

**优化方法为更换为 rg 读取文件内容** **并禁用cmd**

# 第一步：去github 下载 rg

 https://github.com/BurntSushi/ripgrep

下载此版本即可


# 第五步：配置全局提示词 让codex使用rg

在  User\\.codex 目录下新建一个AGENTS.md

放入下面的内容

```
# 工作约定
- 在新增环境依赖之前，请先征得确认。  
- 我的操作系统是 Windows 10。  
- 项目文件的默认编码为 UTF-8。
- 查询内容时可用 PowerShell rg 
- 使用PowerShell，**不要**使用 `cmd`。
```
