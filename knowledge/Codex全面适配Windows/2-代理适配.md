# 第一步：终端代理配置

powershell 输入

```
Set-ExecutionPolicy -Scope LocalMachine RemoteSigned -Force
```

powershell 输入 

```
$profile
```

![image-20260310123731861](./assets/2-%E4%BB%A3%E7%90%86%E9%80%82%E9%85%8D/image-20260310123731861.png)

检查打印出来的文件路径下是否存在该文件，**没有就新建**

在文件中加上如下配置：
（这里我是用的V2rayN 代理，如果你使用Clash Verge，要将端口设置为 7897，其他的代理工具自行查阅代理端口）

```
$Env:HTTP_PROXY="http://127.0.0.1:10808"
$Env:HTTPS_PROXY="http://127.0.0.1:10808"
$Env:ALL_PROXY="http://127.0.0.1:10808"
```

![image-20260310123900876](./assets/2-%E4%BB%A3%E7%90%86%E9%80%82%E9%85%8D/image-20260310123900876.png)

保存并重启 powershell 即可
